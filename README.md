# AI Stock Research Assistant

A financial research tool built with Python and Streamlit that provides real-time stock analysis powered by AI.

## Features
- Company overview and description
- Key valuation metrics (Market Cap, P/E Ratio, EPS)
- 1-year historical stock price chart
- Real-time stock data via Yahoo Finance

## Technologies
- Python
- Streamlit
- yFinance

## How to Run
git clone https://github.com/Niloy19229/ai-stock-research-assistant.git
cd ai-stock-research-assistant
pip install streamlit yfinance
streamlit run app.py

## Usage
Enter any stock ticker (e.g. AAPL, NVDA, MSFT) to get instant research data.
